动态地绑定 HTML 属性或组件 prop 到表达式。例如：`v-bind:title="message"` 绑定元素的 `title` 属性到 Vue 实例的 `message` 属性的值。

常用组件库封装中attribute透传：`v-bind="$attrs"` `v-bind="$listeners"`
